# printThis.js
PrintThis.js let you print the content with the class printThis in it

## Simple Example
```html
<div class="row printThis">
	This is one div to print. You can add a how many class you want
</div>

<div class="printThis">
	This is a second div to print
</div>

<input type="button" value="Print" onclick="printThis()" />
```

## Download/ Installation
Just clone this repository and add the js folder to your project.
Then you have to add this line to the file

```javascript
<script src="js/printThis.js"></script>
```

Also don't forget to add the class printThis to the content to be printed
